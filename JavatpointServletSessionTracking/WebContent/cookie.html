1. Cookie

A cookie is a small piece of information that is persisted between multiple client requests.

We add cookie to the response from the servlet. This response goes back to the client and the cookie is stored in the cache of the client's browser.

When client sends another request, the request comes with the cookie added to it by default.

Thus, we recognize the user.

There are 2 types of cookies in servlets.

i. Persistent Cookie:

It is valid for multiple session . It is removed only if user logout or signout. It is not removed when user closes the browser.

ii. Non-Persistent Cookie:

It is valid for single session only. It is removed each time user closes the browser.


1. To retrieve a cookie,

Cookie cookie[] = request.getCookies();

if(cookie.length > 0)

{ String id = cookie[0].getValue(); }

2. To create a cookie,

Cookie cookie = new Cookie("ID", "2600860");

response.addCookie(cookie);

3. To delete a cookie,

Cookie cookie = new Cookie("ID", "");

response.addCookie(cookie);
